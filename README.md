# Peak Metronome 

A studio-grade, high-precision metronome built for professional musicians.

## High-Precision Engine
Peak uses a **Web Audio API scheduler** paired with a **Web Worker clock** to ensure drift-free, sample-accurate timing even when the browser tab is in the background.

## Key Features
- **Geometric Visualizer**: Visualize rhythms as polygons (Polyrhythms supported).
- **Pendulum Mode**: Classic swing mechanics for traditional feel.
- **AI Practice Coach**: Powered by Gemini 3 Flash. Natural language control (e.g., "Give me a slow blues shuffle").
- **Inner Clock Training**: Randomly mutes beats to help you master internal timing.
- **Tempo Ramp**: Systematic accelerando/ritardando for practice routines.
- **Poly-track Layers**: Run multiple rhythms simultaneously (e.g., 5 over 4).
- **Haptic Feedback**: Mechanical-style haptics for mobile devices.
- **Wake Lock**: Keeps your screen on during practice sessions.

## Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS 4, Motion.
- **Backend**: Express (Gemini API Proxy).
- **AI**: Google Gemini 3 Flash.
